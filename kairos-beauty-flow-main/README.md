# Kairos Estética Integral y Bienestar

Un centro de estética integral en Carabanchel, Madrid que ofrece tratamientos faciales, corporales, depilación, masajes y maquillaje personalizado.

## Características

- Reserva de citas online
- Catálogo de tratamientos
- Información de contacto
- Diseño responsive

## Tecnologías

- React con TypeScript
- Vite
- Tailwind CSS
- React Router
